#  Pranesh Majumder Tirtha — Personal Portfolio Website

Welcome to the repository of my personal portfolio website!  
This site showcases my profile, skills, projects, education, certifications, and ways to connect with me.  
Built with clean UI/UX, responsive design, and smooth scrolling animations.



**Sections Included**
- **Home** — Introduction with profile image and quick navigation  
- **About Me** — Personal background & hobbies  
- **Education** — Beautiful education timeline with color-coded cards  
- **Skills** — Interactive skill chips with hover effects  
- **Projects** — Card-based project showcase  
- **Certifications** — Professional credentials  
- **Resume** — Download / view resume options  
- **Contact** — Social links and contact form
